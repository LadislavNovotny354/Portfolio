A simple project that helps you manage tasks efficiently using Python.  
Built to demonstrate good coding practices and a clean project structure.

## Features
- Add, update, and delete tasks
- Mark tasks as completed
- Search tasks by keyword
- Export tasks to a CSV file
- Simple and clean command-line interface
